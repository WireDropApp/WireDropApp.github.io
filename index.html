WireDrop

Transfer files between iOS and macOS via USB.

Overview

WireDrop lets you transfer files between your iOS and macOS devices via USB cable. This can be useful in situations where other means of transferring files are not possible, such as, if you have WiFi or Bluetooth disabled on your device(s).

  • Supports transferring up to 10 files at a time.
  • HEVC/HEIC file formats are converted to H264/JPG formats.
  • Control via Mac's Menu Bar.
System Requirements

Verify that your setup meets the following requirements:

  • iOS device running iOS 16.0 or later.
  • Mac running macOS 14.0 (Sonoma) or later.
Installation
  1. Install the WireDrop iOS app via the App Store.
  2. Download and run the Mac app Installer package.
Using the app

With everything installed it is very easy to get started:

iOS -> macOS

  1. Make sure that WireDrop app is running on your Mac.
  2. Attach the USB cable.
  3. When sharing a file from iOS, WireDrop will appear as an option on the share sheet (if the file type is supported).
  4. Select WireDrop and wait for your Mac device to appear in the list.
  5. Tap your device to start the transfer.
  6. Files are automatically saved to your Mac's Downloads folder.

macOS -> iOS

  1. Open the WireDrop app on iOS to foreground. It will now start waiting for a transfer.
  2. Attach the USB cable.
  3. On your Mac, select the WireDrop icon from the Status Bar, then click "Send File...". A transfer window will now open.
  4. Drag-and-drop your file(s) to the transfer window to start the transfer.
  5. Once the transfer is complete, the iOS app will open a share sheet where you can select where you want to save the file(s).
